Creating postgre db :
1. sudo -u postgres psql (enter)
2. create database test_tg; (create)
3. \list (see all exists db)
4. DROP DATABASE IF EXISTS test_tg; (delete test_tg)

Create user :
1. CREATE USER zhenyaVo;
2. \du
3. create user zhenya with encrypted password '123';
4. grant all privileges on database test_tg to zhenya;

Path to postgre config :
1. SHOW config_file;
